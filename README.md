# LIFE748-assessment-2
Integrated bioinformatics analysis of E. coli genome assembly, differential gene expression, machine learning classification, and AlphaFold-based structural modelling.

## Overview
This repository contains the code, figures, and report for LIFE748 Assessment 2. The project integrates genome assembly benchmarking, functional annotation, differential expression analysis, machine learning, and structural bioinformatics in *Escherichia coli*.

## Repository structure
- `report/` final submitted report
- `code/` analysis scripts and Quarto file
- `figures/` figures used in the report
- `results/` output tables from machine learning analysis
- `supplementary/` GAI declaration and prompt history

## Methods summary
- Genome assembly benchmarking with Flye and QUAST-derived metrics
- Genome annotation comparison using Prokka and Bakta
- Differential expression analysis using DESeq2
- Machine learning using logistic regression, LDA, and SVM
- Structural analysis using AlphaFold models
